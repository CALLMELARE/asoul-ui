> Vue3 UI library , for ASoul

### Get Started

#### Install

```bash
npm install asoul-ui
```

or

```bash
cnpm install asoul-ui
```

#### Import

```js
// main.js or main.ts
import { createApp } from "vue";
import App from "./App.vue";
import Asoul from "asoul-ui";

createApp(App).use(Asoul).mount("#app");
```
